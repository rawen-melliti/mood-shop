# Mood Shop 🛒
Mood Shop est une application web e-commerce développée en équipe avec PHP et Symfony.
## 🚀 Technologies utilisées
- PHP
- Symfony
- HTML
- CSS
- SQL / Base de données
## 🎯 Objectif du projet
Créer une plateforme web simple permettant de présenter et gérer des produits dans une boutique en ligne.
## 👥 Travail en équipe
Ce projet a été réalisé en collaboration avec une équipe.
